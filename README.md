# todo-vue
App Todo on Vue 3 you can drag and drop every todo, change theme light/dark, responsive design and filter active/complete/all

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
